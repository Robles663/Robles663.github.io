**Descripcion del proyecto:**
Este proyecto tiene como objetivo practicar y demostrar habilidades clave en el análisis de datos.
El proyecto incluye la creación y gestión de un entorno virtual de Python, así como la instalación de las librerías necesarias para el desarrollo de una aplicación web que permite interactuar con los datos.

Aunque se proporciona un dataset de anuncios de venta de coches, el enfoque principal del proyecto no está en el análisis de datos, sino en la estructura, organización y despliegue de una aplicación funcional.

## Objectivos

* Crear y gestionar un entorno virtual en Python
* Aplicar buenas prácticas de organización del código.
* Desarrollar una aplicación web funcional.

## Tecnologias utilizadas
* Python 3
* Streamlit
* Plotly Express
* VirtualEnv
